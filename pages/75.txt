</div><div class="mw-content-ltr mw-parser-output" lang="en" dir="ltr"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Web crawler used by MSN Search (2004–2010)</div>
<p><b>Msnbot</b> was a <a href="/wiki/Web_crawler" title="Web crawler">web-crawling robot</a> (type of <a href="/wiki/Internet_bot" title="Internet bot">internet bot</a>), deployed by <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> to collect documents from the web to build a searchable index for the <a href="/wiki/MSN_Search" class="mw-redirect" title="MSN Search">MSN Search engine</a>. It went into beta in 2004, and had full public release in 2005. The month of October 2010 saw the official retirement of <b>msnbot</b> from most active web crawling duties and its replacement by <a href="/wiki/Bingbot" title="Bingbot">bingbot</a>.<sup id="cite&#95;ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
</p><p>As of September 2015, <b>msnbot</b> was still active from the <a href="/wiki/Microsoft" title="Microsoft">Microsoft</a> and the <a href="/wiki/Bing_(search_engine)" class="mw-redirect" title="Bing (search engine)">Bing</a> webmaster help &amp; howto documentation still indicated that msnbot was active (but that it would retire soon).<sup id="cite&#95;ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> The verification tool for bingbot<sup id="cite&#95;ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup> previously did not recognise <b>msnbot</b> IP addresses. A test executed on 2016-02-22 resulted in a yes: "Verdict for IP address 157.55.39.150: Yes - this IP address is a verified Bingbot IP address. Name: msnbot-157-55-39-150.search.msn.com."<sup id="cite&#95;ref-4" class="reference"><a href="#cite_note-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup>
</p>
