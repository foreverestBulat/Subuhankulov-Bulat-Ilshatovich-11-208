<p>From the point of view of a given web resource (<a href="/wiki/Referent" title="Referent">referent</a>), a <b>backlink</b> is a regular hyperlink on another web resource (the <a href="/wiki/Referrer" class="mw-redirect" title="Referrer">referrer</a>) that points to the referent.<sup id="cite&#95;ref-ASIST&#95;1-0" class="reference"><a href="#cite_note-ASIST-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup> A <i>web resource</i> may be (for example) a <a href="/wiki/Website" title="Website">website</a>, <a href="/wiki/Web_page" title="Web page">web page</a>, or <a href="/wiki/Web_directory" title="Web directory">web directory</a>.<sup id="cite&#95;ref-ASIST&#95;1-1" class="reference"><a href="#cite_note-ASIST-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
</p><p>A backlink is a <a href="/wiki/Reference" title="Reference">reference</a> comparable to a <a href="/wiki/Citation" title="Citation">citation</a>.<sup id="cite&#95;ref-2" class="reference"><a href="#cite_note-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> The quantity, quality, and relevance of backlinks for a <a href="/wiki/Web_page" title="Web page">web page</a> are among the factors that search engines like <a href="/wiki/Google" title="Google">Google</a> evaluate in order to estimate how important the page is.<sup id="cite&#95;ref-3" class="reference"><a href="#cite_note-3"><span class="cite-bracket">&#91;</span>3<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-Lingras&#95;Akerkar&#95;4-0" class="reference"><a href="#cite_note-Lingras_Akerkar-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup> <a href="/wiki/PageRank" title="PageRank">PageRank</a> calculates the score for each web page based on how all the web pages are connected among themselves, and is one of the variables that <a href="/wiki/Google_Search" title="Google Search">Google Search</a> uses to determine how high a web page should go in <a href="/wiki/Search_engine_results_page" title="Search engine results page">search results</a>.<sup id="cite&#95;ref-5" class="reference"><a href="#cite_note-5"><span class="cite-bracket">&#91;</span>5<span class="cite-bracket">&#93;</span></a></sup> This <a href="/wiki/Weighting" class="mw-redirect" title="Weighting">weighting</a> of backlinks is analogous to <a href="/wiki/Citation_analysis" title="Citation analysis">citation analysis</a> of books, <a href="/wiki/Scholarly_paper" class="mw-redirect" title="Scholarly paper">scholarly papers</a>, and <a href="/wiki/Academic_journal" title="Academic journal">academic journals</a>.<sup id="cite&#95;ref-ASIST&#95;1-2" class="reference"><a href="#cite_note-ASIST-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup><sup id="cite&#95;ref-Lingras&#95;Akerkar&#95;4-1" class="reference"><a href="#cite_note-Lingras_Akerkar-4"><span class="cite-bracket">&#91;</span>4<span class="cite-bracket">&#93;</span></a></sup> A Topical PageRank has been researched and implemented as well, which gives more weight to backlinks coming from the page of a same topic as a target page.<sup id="cite&#95;ref-6" class="reference"><a href="#cite_note-6"><span class="cite-bracket">&#91;</span>6<span class="cite-bracket">&#93;</span></a></sup>
</p><p>Some other words for <i>backlink</i> are <b>incoming link</b>, <b>inbound link</b>, <b>inlink</b>, <b>inward link</b>, and <b>citation</b>.<sup id="cite&#95;ref-ASIST&#95;1-3" class="reference"><a href="#cite_note-ASIST-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup>
</p>
<div class="mw-heading mw-heading2"><h2 id="Backlinks_and_search_engines">Backlinks and search engines</h2></div>
<style data-mw-deduplicate="TemplateStyles:r1320445320">.mw-parser-output .hatnote{font-style:italic}.mw-parser-output div.hatnote{padding-left:1.6em;margin-bottom:0.5em}.mw-parser-output .hatnote i{font-style:normal}.mw-parser-output .hatnote+link+.hatnote{margin-top:-0.5em}@media print{body.ns-0 .mw-parser-output .hatnote{display:none!important}}</style><div role="note" class="hatnote navigation-not-searchable">Main article: <a href="/wiki/Link_building" title="Link building">Link building</a></div>
<p><a href="/wiki/Search_engine" title="Search engine">Search engines</a> often use the number of backlinks that a website has as one of the most important factors for determining that website's search engine ranking, popularity and importance.<sup id="cite&#95;ref-7" class="reference"><a href="#cite_note-7"><span class="cite-bracket">&#91;</span>7<span class="cite-bracket">&#93;</span></a></sup> <a href="/wiki/Google" title="Google">Google</a>'s description of its <a href="/wiki/PageRank" title="PageRank">PageRank</a> system (January 1998), for instance, noted that "<a href="/wiki/Google" title="Google">Google</a> interprets a link from page A to page B as a vote, by page A, for page B."<sup id="cite&#95;ref-8" class="reference"><a href="#cite_note-8"><span class="cite-bracket">&#91;</span>8<span class="cite-bracket">&#93;</span></a></sup> Knowledge of this form of search engine rankings has fueled a portion of the <a href="/wiki/Search_engine_optimization" title="Search engine optimization">search engine optimization</a> (SEO) industry commonly termed <a href="/wiki/Linkspam" class="mw-redirect" title="Linkspam">linkspam</a>, where a company attempts to place as many inbound links as possible to their site regardless of the context of the originating site.<sup id="cite&#95;ref-ent&#95;9-0" class="reference"><a href="#cite_note-ent-9"><span class="cite-bracket">&#91;</span>9<span class="cite-bracket">&#93;</span></a></sup> In January 2017, Google launched <a href="/wiki/Google_Penguin" title="Google Penguin">Penguin 4</a> update which devalued such link spam practices.<sup id="cite&#95;ref-ent&#95;9-1" class="reference"><a href="#cite_note-ent-9"><span class="cite-bracket">&#91;</span>9<span class="cite-bracket">&#93;</span></a></sup>
</p><p>The significance of search engine rankings is high, and it is regarded as a crucial parameter in online business and the <a href="/wiki/Conversion_marketing" title="Conversion marketing">conversion rate</a> of visitors to any website, particularly when it comes to <a href="/wiki/Online_shopping" title="Online shopping">online shopping</a>.<sup id="cite&#95;ref-10" class="reference"><a href="#cite_note-10"><span class="cite-bracket">&#91;</span>10<span class="cite-bracket">&#93;</span></a></sup> Blog commenting, guest blogging, article submission, <a href="/wiki/Press_release" title="Press release">press release</a> distribution, social media engagements, and forum posting can be used to increase backlinks.
</p><p>Websites often employ SEO techniques to increase the number of backlinks pointing to their website. Some methods are free for use by everyone whereas some methods, like <a href="/wiki/Link_bait" class="mw-redirect" title="Link bait">linkbaiting</a>, require quite a bit of planning and marketing to work.<sup id="cite&#95;ref-11" class="reference"><a href="#cite_note-11"><span class="cite-bracket">&#91;</span>11<span class="cite-bracket">&#93;</span></a></sup> There are also paid techniques to increase the number of backlinks to a target site. For example, private <a href="/wiki/Blog_network" class="mw-redirect" title="Blog network">blog networks</a> can be used to purchase backlinks. It has been estimated that the average cost of buying a link in 2019 was $291.55 and $391.55, when marketing blogs were excluded from the calculation.<sup id="cite&#95;ref-12" class="reference"><a href="#cite_note-12"><span class="cite-bracket">&#91;</span>12<span class="cite-bracket">&#93;</span></a></sup>
</p><p>There are several factors that determine the value of a backlink. Backlinks from <a href="/wiki/Domain_authority" title="Domain authority">authoritative</a> sites on a given topic are highly valuable. If both sites and pages have content geared toward the topic, the backlink is considered relevant and believed to have strong influence on the search engine rankings of the <a href="/wiki/Webpage" class="mw-redirect" title="Webpage">web page</a> granted the backlink.<sup id="cite&#95;ref-13" class="reference"><a href="#cite_note-13"><span class="cite-bracket">&#91;</span>13<span class="cite-bracket">&#93;</span></a></sup> A backlink represents a favorable 'editorial vote' for the receiving webpage from another granting webpage. Another important factor is the anchor text of the backlink. <a href="/wiki/Anchor_text" title="Anchor text">Anchor text</a> is the descriptive labeling of the hyperlink as it appears on a <a href="/wiki/Web_page" title="Web page">web page</a>.<sup id="cite&#95;ref-14" class="reference"><a href="#cite_note-14"><span class="cite-bracket">&#91;</span>14<span class="cite-bracket">&#93;</span></a></sup> Search engine bots (i.e., spiders, crawlers, etc.) examine the anchor text to evaluate how relevant it is to the content on a webpage. Backlinks can be generated by submissions, such as directory submissions, forum submission, <a href="/wiki/Social_bookmarking" title="Social bookmarking">social bookmarking</a>, business listing, blog submissions, etc. Anchor text and webpage content congruency are highly weighted in <a href="/wiki/Search_engine_results_page" title="Search engine results page">search engine results page</a> (SERP) rankings of a webpage with respect to any given keyword query by a search engine user.<sup id="cite&#95;ref-15" class="reference"><a href="#cite_note-15"><span class="cite-bracket">&#91;</span>15<span class="cite-bracket">&#93;</span></a></sup>
</p><p>Changes to the algorithms that produce search engine rankings can place a heightened focus on relevance to a particular topic. While some backlinks might be from sources containing highly valuable metrics, they could also be unrelated to the consumer's query or interest.<sup id="cite&#95;ref-16" class="reference"><a href="#cite_note-16"><span class="cite-bracket">&#91;</span>16<span class="cite-bracket">&#93;</span></a></sup> An example of this would be a link from a popular shoe blog (with valuable metrics) to a site selling vintage pencil sharpeners. While the link appears valuable, it provides little to the consumer in terms of relevance.
</p>
