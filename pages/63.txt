</div><div class="mw-content-ltr mw-parser-output" lang="en" dir="ltr"><div class="shortdescription nomobile noexcerpt noprint searchaux" style="display:none">Software acting on behalf of a user</div>
<style data-mw-deduplicate="TemplateStyles:r1320445320">.mw-parser-output .hatnote{font-style:italic}.mw-parser-output div.hatnote{padding-left:1.6em;margin-bottom:0.5em}.mw-parser-output .hatnote i{font-style:normal}.mw-parser-output .hatnote+link+.hatnote{margin-top:-0.5em}@media print{body.ns-0 .mw-parser-output .hatnote{display:none!important}}</style><div role="note" class="hatnote navigation-not-searchable">Not to be confused with <a href="/wiki/User-Agent_header" title="User-Agent header">User-Agent header</a>.</div>
<p>On <a href="/wiki/World_Wide_Web" title="World Wide Web">the Web</a>, a <b>user agent</b> is a <a href="/wiki/Software_agent" title="Software agent">software agent</a> responsible for retrieving and facilitating <a href="/wiki/End_user" title="End user">end-user</a> interaction with Web content.<sup id="cite&#95;ref-1" class="reference"><a href="#cite_note-1"><span class="cite-bracket">&#91;</span>1<span class="cite-bracket">&#93;</span></a></sup> This includes all <a href="/wiki/Web_browser" title="Web browser">web browsers</a>, such as <a href="/wiki/Google_Chrome" title="Google Chrome">Google Chrome</a> and <a href="/wiki/Safari_(web_browser)" title="Safari (web browser)">Safari</a>, some <a href="/wiki/Email_client" title="Email client">email clients</a>, standalone <a href="/wiki/Download_manager" title="Download manager">download managers</a> like <a href="/wiki/Youtube-dl" title="Youtube-dl">youtube-dl</a>, and other <a href="/wiki/Command-line" class="mw-redirect" title="Command-line">command-line</a> utilities like <a href="/wiki/CURL" title="CURL">cURL</a>.<sup id="cite&#95;ref-mdn&#95;2-0" class="reference"><a href="#cite_note-mdn-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup>
</p><p>The user agent is the client in a <a href="/wiki/Client%E2%80%93server_model" title="Client–server model">client–server system</a>. The <a href="/wiki/HTTP" title="HTTP">HTTP</a> <a href="/wiki/User-Agent_header" title="User-Agent header">User-Agent header</a> is intended to clearly identify the agent to the server.<sup id="cite&#95;ref-mdn&#95;2-1" class="reference"><a href="#cite_note-mdn-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> However, this header can be omitted or <a href="/wiki/User_agent_spoofing" class="mw-redirect" title="User agent spoofing">spoofed</a>,<sup id="cite&#95;ref-mdn&#95;2-2" class="reference"><a href="#cite_note-mdn-2"><span class="cite-bracket">&#91;</span>2<span class="cite-bracket">&#93;</span></a></sup> so some websites use other detection methods.
